# Developer scripts


## start_prod
Avvia l'applicazione facendo in modo che questa si agganci ai servizi API backend presenti su AWS
Start the application with connection to the AWS backend API services.

## start_dev
Avvia l'applicazione facendo in modo che questa si aggancio ai servizi API backend locali (localhost:50001)
Start the application with connection to local API (localhost:50001)
